# 更新日志

## [0.2.0] - 2026-02-10

### 新增
- ✨ 引入命名空间导出，提供更清晰的模块结构
  - `utils` 命名空间：包含所有工具函数
  - `components` 命名空间：包含所有 Vue 组件
- ✨ 新增多个日期工具函数：
  - `getRelativeTime()` - 获取相对时间描述
  - `isToday()` - 判断是否为今天
  - `isThisWeek()` - 判断是否为本周
  - `getDateRange()` - 获取日期范围
- ✨ 创建 `TableColumn` 组件（基于 Element Plus 二次封装）
  - 支持日期格式化
  - 支持状态标签
  - 支持自定义格式化
  - 支持插槽扩展
